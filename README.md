## Usage

Create the schema from the MySQL shell:

```bash
mysql -u username -p database_name < ./db/schema.sql
```

Seed the database from the command line:

```bash
npm run seed
```

Start the application’s server:

```bash
npm run start
```

Start the application’s server in development:

```bash
npm run watch
```
